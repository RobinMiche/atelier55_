# Ateliér55

Webová prezentace pro Ateliér55 – kreativní prostor pro fotografii, film a produkci.

## Obsah projektu

- hlavní stránka
- stránka ateliéru s galerií
- stránka techniky k zapůjčení
- rezervační poptávka
- kontaktní stránka
- jednoduchý košík techniky přes localStorage
- GitHub Pages auto-deploy přes GitHub Actions
